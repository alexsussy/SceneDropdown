# SceneDropdown
A Unity toolbar dropdown for quick scene access.  
Hold shift to load additively, and there's a context menu toggle to either show scenes project-wide, or just in your build settings.

<img width="451" height="130" alt="screenshot" src="https://github.com/user-attachments/assets/5230710d-bdbb-4359-99bf-bca52ff0f7a9" /></br>
Drop the script into a `Editor/` folder, right click the Unity toolbar and select `Scene Dropdown` to make it show up.
